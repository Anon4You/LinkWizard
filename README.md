### 🚀 Installation

**From Source**
```bash
git clone https://github.com/Anon4You/LinkWizard.git
cd LinkWizard
pip install -r requirements.txt
```

[**Termux Void Repo**](https://termuxvoid.github.io/)
```bash
pkg install linkwizard
```

### 💻 Getting Started

Run the tool with your search term. Use `--help` to view all available options.

```bash
linkwizard --help
```

### 🎨 Features

*   **Zero Config** — No API keys or domain lists required.
*   **Privacy Friendly** — Uses DuckDuckGo HTML scraping.
*   **HTML Reports** — Auto-generates clean, card-based reports.
*   **Termux Ready** — Optimized for mobile environments.
